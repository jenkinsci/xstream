Modules

XStream project consists of several modules:
  1. xstream - the core module
  2. xstream-hibernate - the Hibernate support mdule
  3. xstream-distribution - the module with the documentation and distribution definition
When you clone the Git repository you'll get all the modules.

Browser

You can use the browser to navigate within the sources online.

Source Control Manager

XStream uses Git as the SCM. See Git website for details and documentation on IDE integration.

Access

Clone the repository at GitHub. See GitHub documentation about the different possibilities to access the source code.
